# JavaScript Projects

This repository contains JavaScript projects split into two categories:
- Core JavaScript (console-based logic)
- DOM Projects (browser-based interaction)

## Core JavaScript

Focus on programming fundamentals without UI or DOM.

Topics:
- variables and data types
- functions
- loops and conditions
- arrays and strings
- algorithms and logic challenges

Examples:
- Boolean checks
- Factorial calculator
- Leap year checker
- String tools
- Array manipulation
- Random selection programs

## DOM Projects

Focus on browser interaction using HTML, CSS, and JavaScript.

Topics:
- DOM manipulation
- event handling
- form handling
- dynamic UI updates

Examples:
- interactive buttons
- form validation
- mini web apps

## Structure

- /core-js
- /dom-projects
